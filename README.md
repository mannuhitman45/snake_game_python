# Snake Game 🐍

A classic Snake Game developed using Python and Tkinter as part of **Task 3** of the **InternPe Python Programming Internship**.

## Features
- Smooth snake movement
- Keyboard controls
- Random food generation
- Score tracking
- Snake growth
- Wall collision detection
- Self-collision detection
- Game Over screen

## Technologies Used
- Python
- Tkinter
